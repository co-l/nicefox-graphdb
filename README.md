# NiceFox GraphDB

> **Warning**: This project is under active development and not yet production-ready.

## Cypher Support

| Keyword | Status |
|---------|--------|
| `CREATE` | Supported |
| `MATCH` | Supported |
| `MERGE` | Supported |
| `WHERE` | Supported |
| `SET` | Supported |
| `DELETE` | Supported |
| `DETACH DELETE` | Supported |
| `RETURN` | Supported |
| `AS` (aliases) | Supported |
| `LIMIT` | Supported |
| `COUNT` | Supported |
| `AND` / `OR` / `NOT` | Supported |
| `IS NULL` / `IS NOT NULL` | Supported |
| `CONTAINS` / `STARTS WITH` / `ENDS WITH` | Supported |
| `ORDER BY` | Supported |
| `SKIP` | Supported |
| `DISTINCT` | Supported |
| `OPTIONAL MATCH` | Supported |
| `WITH` | Not supported |
| `UNION` | Not supported |
| `UNWIND` | Not supported |
| `CASE` | Not supported |
| `EXISTS` | Not supported |
| `CALL` | Not supported |
| `SUM` / `AVG` / `MIN` / `MAX` / `COLLECT` | Not supported |
| Variable-length paths (`*1..3`) | Not supported |
